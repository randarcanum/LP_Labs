Двуполостным гиперболоидом называется поверхность, которая в некоторой декартовой системе координат определяется каноническим уравнением

Геометрические свойства двуполостного гиперболоида:

  1. Каждая горизонтальная плоскость z = h при |h| < c не пересекает гиперболоид, при |h| = c имеет единственную общую точку с гиперболоидом ((0, 0, c) при h = c и (0, 0, −c) при h = −c). При |h| > c плоскость z = h пересекает гиперболоид по эллипсу.
  2. Вертикальные плоскости x = 0 и y = 0 пересекают гиперболоид по гиперболам, у которых действительной осью является ось Oz.
  3. Поверхность состоит из двух симметричных полостей, имеющих форму неограниченных чаш, расположенных в полупространствах z ≥ c и z ≤ −c.
  4. Ось Oz называется продольной осью гиперболоида, оси Ox и Oy называются поперечными осями гиперболоида.
