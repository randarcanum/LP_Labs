Эллиптическим параболоидом называется поверхность, которая в некоторой декартовой системе координат определяется каноническим уравнением

Геометрические свойства эллиптического параболоида:

  1. Каждая плоскость z = h при h < 0 не пересекает параболоид, при h = 0 имеет с ним единственную общую точку O(0, 0, 0), при h > 0 пересекает параболоид по эллипсу.
  2. Плоскости x = 0 и y = 0 пересекают параболоид по параболам.
  3. Поверхность состоит из одной полости, имеющей форму неограниченной чаши, расположенной в полупространстве z ≥ 0.
  4. Ось Oz называется осью эллиптического параболоида, точка O(0, 0, 0) - его вершиной.
