Однополостным гиперболоидом называется поверхность, которая в некоторой декартовой системе координат определяется каноническим уравнением

Геометрические свойства однополостного гиперболоида:

  1. Каждая горизонтальная плоскость z = h пересекает гиперболоид по эллипсу. Эллипс получающийся при h = 0, называется горловым эллипсом гиперболоида.
  2. Вертикальные плоскости x = 0 и y = 0 пересекают гиперболоид по гиперболам.
  3. Ось Oz называется продольной осью гиперболоида, оси Ox и Oy называются поперечными осями гиперболоида.
